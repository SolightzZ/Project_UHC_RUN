
ใบอนุญาตการใช้งาน

ส่วนเสริม Sleeplite UHC Run สำหรับ Minecraft Bedrock Edition
ใช้ได้เฉพาะบน Sleeplite Server (SMP) เท่านั้น

ห้ามแจกจ่าย ดัดแปลง หรือใช้ในเชิงพาณิชย์โดยไม่ได้รับอนุญาต
ห้ามแอบอ้างเป็นเจ้าของหรือผู้พัฒนา
ไม่มีการรับประกัน ส่วนเสริมนี้ให้ใช้ฟรีตามสภาพ
หากต้องการขอสิทธิ์ใช้งานเพิ่มเติม ติดต่อ Sleeplite Server ผ่าน Discord

1. ขอบเขตการอนุญาต
ข้อตกลงนี้ให้สิทธิ์คุณใช้งานส่วนเสริม Sleeplite UHC Run บน Sleeplite Server (SMP) เท่านั้น โดยเป็นการอนุญาตแบบจำกัด ไม่ผูกขาด โอนไม่ได้ และเพิกถอนได้

2. ข้อจำกัด
คุณไม่ได้รับอนุญาตให้ดำเนินการดังต่อไปนี้:
แจกจ่ายซ้ำ: ห้ามอัปโหลด แชร์ หรือเผยแพร่ส่วนเสริมนี้บนแพลตฟอร์มอื่นๆ โดยไม่ได้รับอนุญาต
แก้ไขและแจกจ่ายซ้ำ: ห้ามดัดแปลง ทำวิศวกรรมย้อนกลับ หรือเผยแพร่เวอร์ชันที่แก้ไขของส่วนเสริมนี้
การแอบอ้างเป็นเจ้าของ: ห้ามใช้ส่วนเสริมนี้ในเชิงพาณิชย์หรืออ้างว่าเป็นเจ้าของหรือผู้สร้างโดยไม่ได้รับอนุญาต

3. ความเป็นเจ้าของ
ส่วนเสริมนี้ รวมถึงทรัพย์สินทางปัญญาที่เกี่ยวข้องทั้งหมด เป็นกรรมสิทธิ์ของ SolightzZ และ Sleeplite Server (SMP)

4. การติดต่อ
หากต้องการสอบถามเพิ่มเติมหรือต้องการขออนุญาตใช้งานเพิ่มเติม กรุณาติดต่อ Sleeplite Server (SMP) ผ่าน Discord: Sleeplite Server

5. การปฏิเสธความรับผิด
ส่วนเสริมนี้ให้ใช้งาน ตามสภาพ ("as is") โดยไม่มีการรับประกันใดๆ ไม่ว่าโดยชัดแจ้งหรือโดยปริยาย

6. ข้อจำกัดความรับผิด
SolightzZ และ Sleeplite Server (SMP) จะไม่รับผิดชอบต่อความเสียหายใดๆ ที่เกิดจากการใช้งานหรือการไม่สามารถใช้งานส่วนเสริมนี้

7. การเปลี่ยนแปลงข้อตกลง
Sleeplite Server (SMP) ขอสงวนสิทธิ์ในการแก้ไขข้อตกลงนี้ได้ตลอดเวลาโดยไม่ต้องแจ้งให้ทราบล่วงหน้า
